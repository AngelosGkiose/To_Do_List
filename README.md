# To-Do List 

A command-line To-Do List application built with Python.

The application allows users to manage daily tasks by adding, editing, deleting, and marking them as completed. Tasks are stored in a JSON file, so they are automatically saved between program runs.

## Features

- View all tasks
- Add new tasks
- Delete existing tasks
- Edit task titles
- Mark tasks as completed
- Prevent duplicate tasks
- Persistent storage using JSON
- Input validation

## Concepts Practiced

- Functions
- Lists
- Dictionaries
- JSON file handling
- Loops
- Conditional statements
- Exception handling (try / except)
- CRUD operations
